Git, GitHub, and Turning in your programming homework

In this thread, you will learn about social coding and the version control system called git, which is the software behind the website github.com. Version control systems are the primary way that all programmers, both professional and amateur, keep track of their software, compare the different versions over time, and share code with one another. It is how future employers or research collaborators will evaluate your coding ability, and it's probably also how you will choose to share your code with the world.

If you already know about git and just want to submit your homework, skip to the bottom of the page.

Here is a tutorial to learn about the basics of git and control.

There are two ways to use git and GitHub, using the command-line interace (git only) which is good for experts, or using the graphical interface for GitHub, which is good for beginners. We describe both ways below. Both ways are completely acceptable for this class.

We will discuss all of these options below.

Explore GitHub

First of all, explore GitHub! Many open source software projects that are behind websites that you use every day are hosted on GitHub. Go to http://github.com/explore and look through the most popular. If you know someone, perhaps a classmate in CSF, who has a github account, ask them for their username and ask them if you can look at their code. For example, my GitHub username is phamp and you can view all of my public repositories here: http://github.com/ppham.

Many famous programmers code on github, such as Linus Torvalds, the creator of Linux and git itself, or Guido van Rossum, the inventor of Python. Try googling for "Linus Torvalds github" or "Guido van Rossum github".

To submit homework

You will submit homework by

  1. adding and committing all the necessary files on your computer that are related to the homework
  2. creating tags with a name like hwx (for example, hw1 for submitting Homework 1,)
  3. pushing the tag (and the master branch) to your github repository.
You must do all of these steps before the homework deadline, or it will be counted as late. Late homework will still be accepted, but it will affect your course evaluation.
